Plugging Your Raspberry Pi Dongle to a Windows Computer and Finding It¶

These directions were very helpful: https://www.circuitbasics.com/raspberry-pi-zero-ethernet-gadget/

Don’t forget to enable SSH capability on your pi by creating a file on the boot drive named SSH. (No file extension.) Not sure how to create a file without a file extension? Create a file in Notepad or Notepad +++ with quotes on both sides of the name; a file extension is not required or included.

Connecting to Your Raspberry Pi¶

Before beginning, make sure that your Raspberry Pi has a fresh install of the operating system and tht you have added a fild to the boot drive named SSH. (The SSH file cannot have a file extension. See above if you do not know how to do this.)

Connect your Raspberry Pi (attachd to the USB dongle) to your computer. (A USB extension cord with a male and female end can make this more convenient.)

You have two options for connecting to your Raspberry Pi, either with Secure Shell (SSH) or via VNC Viewer.

For a comprehensive overview regarding this, see the Pi documentation https://www.raspberrypi.com/documentation/computers/remote-access.html

To connect to the Pi using SSH, make sure you have PuTTY. You can download it here https://www.chiark.greenend.org.uk/~sgtatham/putty/latest.html. Open putty and connect to raspberrypi.local. You will be prompted to entering the username (pi, unless you have changed this) and the password (raspberry if, again, you have not changed it). This will provide you with access to your pi via the command line.

To connect with your pi with a GUI (graphical user interface–a desktop), you will need to use VNC Viewer.
